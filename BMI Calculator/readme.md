# BMI Calculator
## Project Title
BMI Calculator using Python
## Project Overview
This project is a simple Body Mass Index (BMI) Calculator developed using Python. It allows users to enter their weight and height, calculates the BMI value, and classifies the result into standard health categories such as Underweight, Normal Weight, Overweight, and Obese.
This project is suitable for beginners who want to understand the basics of Python programming and build a small real-world application.
## Key Features
- Accepts user input for weight and height
- Calculates BMI using a standard mathematical formula
- Displays the BMI result
- Classifies BMI into health categories
- Handles invalid or incorrect user input
- Beginner-friendly and easy to understand
## Technologies Used
- Python
## Concepts Used
- Functions
- Conditional Statements
- User Input Handling
- Exception Handling
- Mathematical Calculations
## BMI Formula
BMI = Weight (kg) / Height² (m²)
## BMI Categories
- Underweight: Less than 18.5
- Normal Weight: 18.5 to 24.9
- Overweight: 25 to 29.9
- Obese: 30 and above
## Project Workflow
1. User enters weight in kilograms
2. User enters height in meters
3. Program calculates BMI
4. Program displays BMI result
5. Program shows BMI category
