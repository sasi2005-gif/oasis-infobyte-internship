# Voice Assistant
## Project Title
Voice Assistant using Python
## Project Overview
This project is a Python-based **Voice Assistant** with a graphical interface (Tkinter). It listens to voice commands from the user, responds using speech output, and can perform tasks such as telling the time and date, opening websites, and simple conversation.
This project is suitable for beginners and intermediate learners who want to understand Python GUI development, threading, and voice interaction.
## Key Features
- Accepts voice input from the user using the microphone
- Responds with voice output using text-to-speech
- Provides current time and date
- Performs Google web searches
- Opens applications like WhatsApp
- Handles invalid commands gracefully
- Displays conversation in a GUI scrollable text box
## Technologies Used
- Python
- Tkinter for GUI
- SpeechRecognition for voice input
- Pyttsx3 for voice output
- PyAudio for microphone access
- Webbrowser module for web actions
## Concepts Used
- Functions
- Conditional statements
- Threading for responsive GUI
- Voice recognition and text-to-speech
- GUI design with Tkinter
- Exception handling
## Voice Assistant Commands
- "Hello" – Greet the assistant
- "How are you?" – Conversational response
- "Time" – Get current time
- "Date" – Get current date
- "Search [query]" – Search Google
- "Open YouTube" or "Open Google" – Opens website
- "Exit" or "Stop" – Close assistant
## Project Workflow
1. Run the program to open the GUI
2. The assistant announces it is ready
3. User speaks a command
4. Program recognizes the command
5. Program performs the task and responds via voice and GUI
6. The user can continue giving commands or exit
## Learning Outcomes
By completing this project, learners can understand:
- Python GUI development with Tkinter
- How to integrate voice recognition and text-to-speech
- Threading for responsive applications
- Handling exceptions and invalid inputs
- Automating tasks like web search and opening applications
